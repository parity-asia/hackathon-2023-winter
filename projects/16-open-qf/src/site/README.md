# OpenQF site

## Prerequisites

- [Node.js 18.17](https://nodejs.org/) or later
- [yarn](https://yarnpkg.com)

## Env Config

Copy `.env.example` to `.env`

```sh
cp .env.example .env
```

## Install dependencies

```sh
yarn
```

## Run development

```sh
yarn dev
```

Open http://127.0.0.1:3000 with your browser to see the result.

## Run production

```sh
yarn build
yarn start
```

Open http://127.0.0.1:3000 with your browser to see the result.

## Run test

```sh
yarn test
```
